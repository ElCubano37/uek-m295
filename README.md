# uek-m295

> Projektarbeit für das ÜK Modul 295

## Aufsetzen und starten der APP

Benutze den Befehl `npm install` um die nötigen Module zu installieren. 

Zum starten der App benutze den Befehl: `node main.js`. Stelle sicher, dass du im richtigen Verzeichnis bist. 

## OpenAPI und Swagger

Um die OpenAPI Dokumentation zu öffnen muss zuerst die App gestartet werden. Anschliessend kann man folgenden Link öffnen: [{http://localhost:3000/api-docs}]
